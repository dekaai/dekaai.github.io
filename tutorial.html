Tutorial Lengkap Membuat Domain dengan Cloudflare

Panduan step-by-step dari pembelian domain hingga deployment website dengan GitHub Pages dan Cloudflare

Persiapan Awal

1

Membuat Kode HTML

Siapkan file HTML untuk website Anda. Contoh kode dasar:

<!DOCTYPE html>
<html lang="id">
<head>
    <meta charset="UTF-8">
    <meta name="viewport" content="width=device-width, initial-scale=1.0">
    <title>Website Saya</title>
</head>
<body>
    <h1>Selamat Datang</h1>
</body>
</html>

Simpan sebagai index.html

Membeli Domain

2

Beli Domain di IDWebhost

Kunjungi IDWebhost untuk membeli domain.

Catatan

Tunggu hingga domain aktif (1-24 jam).

Setup GitHub

3

Buat Repository

Buat repository baru dengan format: username-github-io

Buat Repository

Cloudflare Setup

4

Hubungkan Domain

Tambahkan domain Anda di Cloudflare dan ganti nameserver di IDWebhost.

Troubleshooting

Solusi Masalah Umum

  • Domain tidak aktif? Tunggu 24 jam
  • Error GitHub? Periksa nama repository
  • Website tidak update? Cek deployment
